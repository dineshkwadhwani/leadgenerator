⚙️ CONFIGURATION (config.json)
═════════════════════════════════════════════════════════════════════════════

{
  "INPUT_FILE": "PIMPRI_CHICHWAD_MNP.xlsx",  ← Change this to your file
  "MAX_WORKERS": 5,                           ← 1-10 (higher = faster/risky)
  "REQUEST_TIMEOUT": 15,                      ← Seconds to wait for response
  "VERBOSE": true                             ← Detailed logging
}

QUICK CHANGES:
• Change file: Edit "INPUT_FILE"
• Run faster: Increase "MAX_WORKERS" (5→10)
• Run stable: Decrease "MAX_WORKERS" (5→3)
• Longer wait: Increase "REQUEST_TIMEOUT" (15→30)
